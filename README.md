# Signature Verification System

A web-based Signature Verification System developed using **Python**, **Django**, and **OpenCV** to verify handwritten signatures using image processing techniques.

## Features

- User authentication
- Signature upload
- Signature verification
- Django admin panel
- Image processing with OpenCV

## Technologies Used

- Python
- Django
- OpenCV
- HTML
- CSS
- SQLite

## Project Structure

```
signature-verification/
│── app/
│── signature_verification/
│── manage.py
│── requirements.txt
```

## Installation

```bash
git clone https://github.com/Afijith-Sreeni/signature-verification-django.git
cd signature-verification
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
